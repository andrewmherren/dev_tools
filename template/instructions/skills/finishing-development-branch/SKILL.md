# Finishing Development Branch

Use this when implementation and review updates are done.

## Procedure

1. Run final verification commands.
2. Confirm review feedback is resolved.
3. Prepare concise merge or PR summary.
4. Clean up temporary artifacts and stale worktrees.

## Exit criteria

- Verification is green or remaining failures are explicitly documented.
- Reviewer-facing summary is complete and accurate.

# Unreal Conventions

- Use Unreal coding standard for C++ class naming.
- Keep generated files out of source control unless required.
- Organize modules by gameplay/system ownership.
- Validate editor and build commands from VS Code tasks.

## Verification Commands

- **Build**: Use VS Code build task or `UnrealBuildTool`
- **Test**: Unreal Automation Testing framework (requires project setup)
- **Editor validation**: Launch Unreal Editor to verify compilation
- Note: Use tasks.json for project-specific build/test commands.

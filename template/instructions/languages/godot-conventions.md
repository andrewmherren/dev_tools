# Godot Conventions

- Keep scene and script naming consistent.
- Favor small reusable scenes over large monolith scenes.
- Store project settings in text mode for reviewability.
- Keep assets and scripts in predictable folders.

## Verification Commands

- **Test**: GDScript unit testing (requires Gut or project-specific framework)
- **Lint**: Limited native tooling; manual review recommended
- **Editor validation**: Open project in Godot editor to check for errors
- Note: Automated verification limited; prioritize manual editor checks.

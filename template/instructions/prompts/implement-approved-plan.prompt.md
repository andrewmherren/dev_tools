name: implement-approved-plan
description: Execute an approved task plan with explicit verification after each task.
---

Implement the approved plan in small, reviewable steps.

Requirements:

- Keep scope aligned to the approved plan.
- Use `docs/design/<feature>.md` and `docs/plans/<feature>-plan.md` as the durable handoff source when available.
- Run verification for each completed task.
- Report command results and any remaining unverified areas.
- Update the plan doc if task order, scope, or verification steps change.
